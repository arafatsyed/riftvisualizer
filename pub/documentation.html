Getting Started


Creating your first map

Before you get started and start using this library, there are a few things you should know and need to do before hand.

1) Install jQuery
Firstly, this library uses jQuery for DOM manipulation so you need to include that in your html file. Put this line below in your html file:

<script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

2) Including Rift Visualizer Library

Next, install the Rift Visualizer library from github. Install the riftvisualizer.js file and pictures folder and put them in the same level in your project folder. After installing, in your html file put:

<script defer type="text/javascript" src = "{yourdirectory}/riftvisualizer.js" ></script>

3) Create an html element with an id.

Put an html element with a unique ID in the area that you want your map. An instance of the library will operate inside this element:

<div id="myFirstMap"> </div>

4) Initialize The Library

In your javascript file for your project, initialize the library. The first argument is the id of the html element you want the library to instantiate in, the second is the type of map you want. There are 3 types: create state, static, and quiz. Let's intialize the map in create state (as seen in the examples). :

const myFirstMap = new RiftVisualizer("#myFirstMap","createState");

 5) Spawn the map

Now you can spawn a map using the rvCreateInterface function. The input is the scale of the map you want from 0 to 1. The largest map is 2000x1423 so you can scale down from that. Let's create a map with a scale of 0.4 (800x570).

myFirstMap.rvCreateInterface(0.4);

 6) Lets add some objects to the map!

You can add as many champions and labels as you want. You can also spawn any of the structures in the game as well as all the jungle monsters.

Note: the positions of the objects you are placing are with respect to the 2000x1423 map even if you have a scaled version. You can use online image coordinate web apps to get specific coordinates or create a map in create mode from the UI and use rvGetGameStateJSON and rvSetGameStateJSON to avoid this (as seen in examples).

//adding champions
myFirstMap.rvSpawnChampion("Jayce","blue",572,162,true);
myFirstMap.rvSpawnChampion("Vi","blue",342,220,true);

//structures
let structures = ["bn1","bn2","btt3","btt2","btt1","bmt3","bmt2","bbt3","bbt2","rn1","rn2","rti","rmi","rbi","rmt3","rmt2","rmt1","rbt3","rbt2","rtt3","rtt2","rtt1","bmi","bti","bbi"]
myFirstMap.rvSpawnStructure(structures);
//monsters
myFirstMap.rvSpawnMonsters(["baron","rbluebuff","rgromp","rwolves","rraptors","bredbuff","bkrugs","braptors","bbluebuff"]);

//add labels
myFirstMap.rvAddMarker("Dragon Dead","#ff3300",1157,917,true);
myFirstMap.rvAddMarker("Blue Team Minions Pushed","aqua",1395,702,true);
myFirstMap.rvAddMarker("ward","yellow",652,287,true);

Your all set! These were the basics. Below is the full list of the API functions. There are a lot more things you can do!


API Functions


Intializing Functions

Constructor

RiftVisualizer("IdOfHTMLElement","stateOfMap")

Example: const myFirstMap = new RiftVisualizer("#myFirstMap","createState");

 Input: Id of the html element you want map to instatiate in, the type of map you want.

Types of Maps:
1) "createState": map with the buttons as seen in example 1. Let's you spawn objects from the front-end.
2) "static": map with no buttons, will spawn an empty image.
3) "quiz": map with no buttons, will spawn an empty image. Use this if you want to create a quiz.

Intialize Map

.rvCreateInterface(scale)

Description: This function spawns a map in the HTML element specified from the constructor call.

Example: myFirstMap.rvCreateInterface(0.4);

Input: the scale of the map you want from 0 to 1. Scaled off of the max size of the original size of the map image 2000x1423.

Setting Permissions

.rvUserDeletePermission(boolean)

Description: This function enables or disables allowing the user to delete items off the map.

Example: (Disabling deleting permssions)
myFirstMap.rvUserDeletePermission(false);

Input: If the input is true, it will allow users to delete objects on the map. If its false, users cannot delete objects. By default, when a map is spawned, the user deleting permission is set to true.


Getting Map Information

Getting Entire Map State

.rvGetGameState()

Description: This function returns all of the spawned objects on the map as an object.

Object: {champions: Array(numChampions), monsters: Array(numMonsters), structures: Array(numStructures), markers: Array(numMarkers)}

Getting Specific Objects

.rvGetSpawnedChampions()

.rvGetSpawnedMonsters()

.rvGetSpawnedStructures()

.rvGetSpawnedMarkers()

Description: This function returns an array of all the spawned objects of the specfied type.

Champion Object: { "championElement": html Element, "name": "champion name", "id": "html element id", "x": "left position", "y": "top position", "team": "color of team", "movable": boolean for it object is movable or not }

Monster Object: { "monsterElement": html element, "index": index in library array, "name": "name of monster" }

Structure Object: { "structure": html element, "index": index in library array, "name": name of structure }

Marker Object: { "markerElement": html element, "text": "custom text", "left": left position, "top": top position, "id": "html id", "movable": boolean if object is movable or not }

Interacting with Champion Objects

Spawning Champions

.rvSpawnChampion("name","team",x,y,movable)

Description: This function lets you spawn a champion on the map at a given position (x,y) with respect to the 2000x1423 size map. The position is scaled down inside the library later. It will spawn a champion with a name and team and there is a boolean that lets you allow the user to move it or not.

Example:
//Movable Champion
gamestateExample.rvSpawnChampion("Jayce","blue",572,162,true);
//Not Movable Champion
gamestateExample.rvSpawnChampion("Jayce","blue",572,162,false);


Input: Takes 2 strings represeting the name and team, two numbers for the x and y position with respect to 2000x1423 map, and a boolean that specifies if the user can move this champion.

Output: Gives you back the champion object (see Getting Specific Objects).

Deleteing Champions

.rvDeleteOneChampion(championObject)

Description: This function lets you delete a spawned champion from the map.

Example:
//championObject retrieved from .rvGetSpawnedChampions()[index]
gamestateExample.rvDeleteOneChampion(championObject);


Input: The champion object of the champion you want to delete. Can be retreived from .rvGetSpawnedChampions()

.rvDeleteAllChampions()

Description: This function lets you delete all spawned champion from the map.

Example:
gamestateExample.rvDeleteAllChampions();


Moving (animating) Champions

.rvMoveChampion(championObject,x,y)

Description: This function lets you animate the movement of a champion.

Example:
//moving champion
jungleExample.rvMoveChampion(jgler,947.5,1020)

Input: The champion object of the champion you want to delete. Can be retreived from .rvGetSpawnedChampions(), the new (x,y) position of the champion with respect to 2000x1423 map


Interacting with Marker Objects

Spawning Markers

.rvAddMarker("label","color",x,y, movable)

Description: This function lets you spawn a marker on the map at a given position (x,y) with respect to the 2000x1423 size map. The position is scaled down inside the library later. It will spawn a marker with a label and color and there is a boolean that lets you allow the user to move it or not.

Example:
//Movable Marker
jungleExample.rvAddMarker("Clear Gromp","yellow",425,510,true);
//Not Movable Marker
jungleExample.rvAddMarker("Clear Gromp","yellow",425,510,false);

Input: Takes 2 strings represeting the name and colour, two numbers for the x and y position with respect to 2000x1423 map, and a boolean that specifies if the user can move this marker.

Output: Gives you back the marker object (see Getting Specific Objects).

Description: This function lets you delete a spawned marker from the map.

Interacting with Structure Objects

Spawning Structures

.rvSpawnStructure(Array of structures)

Description: This function lets you spawn any of the 28 structures in League of Legends on the map.

Example:
let structures = ["bn1","bn2","btt3","btt2","btt1","bmt3","bmt2","bbt3","bbt2","rn1","rn2","rti","rmi","rbi","rmt3","rmt2","rmt1","rbt3","rbt2","rtt3","rtt2","rtt1","bmi","bti","bbi"]
gamestateExample.rvSpawnStructure(structures);

This function takes a string array of the structures you want to spawn. The structures are encoded in a way where each character represents something.

Input:

Normal Towers:
The string format is (team)(lane)t(tier). So for example, if you want to spawn red side top tier 3 tower, the string would be "rtt3".

Nexus towers:
String format is (team)n(1or2). For example, if you wanted to spawn the blue side nexus tower thats on the left, the string would be "bn1"

Inhibitors:
The string format is (team)(lane)i. For example, if you wanted to spawn the red side mid inhibitor, the string would be "rmi"

Deleteing Structures

.rvDeleteStructure(Array of Structures to Delete)

Description: This function lets you delete a spawned structures from the map.

Example:
gamestateExample.rvDeleteStructure(["bn1","bn2","btt3","btt2","btt1","bmt3","bmt2","bbt3","bbt2","rn1","rn2","rti","rmi","rbi","rmt3","rmt2","rmt1","rbt3","rbt2","rtt3","rtt2","rtt1","bmi","bti","bbi"]);


Input: A string array of all of the structures you want to delete off the map

.rvDeleteAllStructures()

Description: This function lets you delete all spawned structures from the map.

Example:
gamestateExample.rvDeleteAllStructures();



Interacting with Monster Objects

Spawning Monsters

.rvSpawnStructure(Array of monsters)

Description: This function lets you spawn any of the 16 monsters in League of Legends on the map.

Example:
gamestateExample.rvSpawnMonsters(["baron","rbluebuff","rgromp","rwolves","rraptors","bredbuff","bkrugs","braptors","bbluebuff","tscuttle","bscuttle"]);


This function takes a string array of the monsters you want to spawn. The structures are encoded in a way where each the starting character represents something for common monsters.

Input:

Epic Monsters:
For Epic Monsters like dragon and baron, the string format is just "dragon" and "baron"

Jungle Monsters:
For the rest of the monsters, the string format is (team)(nameofmonster). So for example if you wanted to spawn red side raptors, the string would be "rraptors"

Scuttle Crab:
If you want to spawn a scuttle crab, the string format is (t or b)scuttle. For example, if you want to spawn top side scuttle, the string would be "tscuttle"

Deleteing Monsters

.rvDeleteMonster(Array of Monsters to Delete)

Description: This function lets you delete spawned monsters from the map.

Example:
gamestateExample.rvDeleteStructure(["baron","bbluebuff","rgromp"]);


Input: A string array of all of the monsters you want to delete off the map

.rvDeleteAllStructures()

Description: This function lets you delete all spawned monsters from the map.

Example:
gamestateExample.rvDeleteAllMonsters();

Quiz Functionality

Add Your Own Quiz

.rvSetQuizZone(championObject,x1,y1,x2,y2)

Description: This function lets you create an interactive quiz on the map. The user has to move champions to the correct position given some prompt. This function lets you create a correct zone for all of the champions that are a part of your quiz. The zone is a rectangle and you need to give the coordinates for the opposite diagonal points.

Example:
//setting the correct zone for the first spawned champion
quizMap.rvSetQuizZone(quizMap.rvGetSpawnedChampions()[0],595,339,891,576);


Input: A champion object and 2 points, (x1,y1),(x2,y2) with respect to 2000x1423, of a rectangle on the map. These points must be on the same digaonal.

.rvEvaluateQuiz()

Description: This function evaluates the position of all of the champion's that are part of the quiz.

Example:
gamestateExamples.rvEvaluateQuiz();

Output: It returns true if all of the champions are in the correct zone or returns an array of the misplaced champions.
