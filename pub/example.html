Rift Visualizer

A League of Legends Map Visualizer Library for Your Web App!

The Rift Visualizer library allows web developers to easily add and visualize different game states for the popular game League of Legends. If you are developing a web app that league of legends players will use to help them with their game (similar to blitz.gg and jungler.gg), then this library is perfect for you. The Rift Visualizer library lets you add a dyanmic and interactive Leauge of Legends map that both the developer and user can modify and interact with.

Learn How To Use


Examples

Create Mode


Create Mode

This can be used both as a user interaction element or a tool to help you develop your web app. This version of the map allows you to access most library functionality easily from the UI. This is very useful while developing with this library since it can make positioning elements (like champions and labels) much easier and faster.

User side:

This free creation mode lets users use most of the functionality of the library. User can:

  • Add champions, drag them around, double click to change team, drag off map to delete
  • Add labels on the map, drag them around, double click to change colour, drag off map to delete.
  • Spawn all the structures and double click them to delete
  • Spawn all the monsters and double click them to delete

Developer side:

The developer can set up a gamestate here without using the library functions (less coding for you!) and then call the function .rvGetGameStateJSON() . This function returns a stringified object (JSON form) of the maps state. The developer can then save the object and pass in the object to the function .rvSetGameStateJSON(gameStateObject) to generate the map that they created in createmode. This speeds up development with the library greatly because you can set up maps quickly without writing a lot of code! Add some things on the map and click on the Get Game State button!


JavaScript Code - Setting Up Create Mode Map

//firstMap is the id of the HTML element this map will intialize in
const newMap = new RiftVisualizer("#firstMap","createState");
//input to this function is the size of the map from 0 to 1
newMap.rvCreateInterface(0.4);


Example Developer Gamestate


Map Visualizer

Here is an example of a developer created game state using library functions. You can set up any game state you want by spawning and moving different champions, spawning towers, and spawning jungle camps (monsters). You can use labels to represent various important things in the game like the position of minion waves and wards, or use it as a caption for something you want to highlight. You can highlight specific objects on the map, for example here the champion Kayn and the monster Baron is highlighted.

Example Use Cases:

  • Blogs about League of Legends can use this library to visualize what they are trying to explain. This one can use this library to highlight different aspects like lanes and towers to explain this game to beginers.
  • This library can be used to visualize live matches as they are being played. Lolesports.com , the official website for this game, displays live stats while a professional game is being played. This library can visualize these stats.
  • Mobalytics is a web app related to League of Legends and they have a challenge section where they give users a challenge to help improve their skills. You can use this library to create different gamestates and challenge/quiz users about what to do next. This can help users learn and improve on the game.

Jungle Clear Use Case (Animations)

Moving Map Objects

This example shows the level 1 starting path for a jungler. This library visualizes what a player needs to do for the first few minutes in a game when they are playing the champion Warwick in the jungle. Hit the button that says next to see the visualization. Notice the user cannot move the champion or label around, or delete the monsters. This is because these objects were created with moveable permission off and the map is in a state where the user cannot delete anything.

Web apps like jungler.gg specializes in showing jungle paths in the early game. This app can use the Rift Visualizer library to make the jungle path they display more dynamic and interactive (scroll to the middle on the website thats linked to see).

JavaScript Code - Moving Objects

    //library set up
    const jungleExample = new RiftVisualizer("#jungleExample","static");
    jungleExample.rvCreateInterface(0.4);
    //Disables Letting Users Delete Objects From Map
    jungleExample.rvUserDeletePermission(false);
    let pos = 0;

    //spawning monsters
    jungleExample.rvSpawnMonsters(["bredbuff","bkrugs","braptors","bwolves","bbluebuff"
    ,"bgromp","tscuttle"]);

    //spawning champion
    let jgler = jungleExample.rvSpawnChampion("Warwick","blue",947,1020, false);

    //adding marker
    jungleExample.rvAddMarker("Start red","yellow",1030,912.5);

    //slide show code. Onclick function for the button prev and next
    function handleOnClick(e){
        if(e==="prev"){
            pos--;
            if(pos<=0){
                pos=7;
            }  
        }
        else{
            pos++;
        }
        moveMarkerExampleTwo();
    }

    
    function moveMarkerExampleTwo(){
        jungleExample.rvDeleteAllMarkers();
        
        if(pos===7){
            pos=0;
            jungleExample.rvAddMarker("Start Red","yellow",1030,912,false);
            //moving champion  
            jungleExample.rvMoveChampion(jgler,947.5,1020,false)
        }
        else if(pos===1){
            jungleExample.rvAddMarker("Clear Raptors","yellow",892.5,767.5,false);
            jungleExample.rvMoveChampion(jgler,975,855,false)
        }
        else if(pos===2){
            jungleExample.rvAddMarker("Clear Wolves","yellow",547.5,705,false);
            jungleExample.rvMoveChampion(jgler,545,755,false)
        }
        else if(pos===3){
            jungleExample.rvAddMarker("Clear Blue Buff","yellow",530,577,false);
            jungleExample.rvMoveChampion(jgler,552.5,630,false)
        }
        else if(pos===4){
            jungleExample.rvAddMarker("Clear Gromp","yellow",425,510,false);
            jungleExample.rvMoveChampion(jgler,480,590,false)
        }
        else if(pos===5){
            jungleExample.rvAddMarker("Clear Scuttle","yellow",627.5,465,false);
            jungleExample.rvMoveChampion(jgler,725,512.5,false)
        }
        else if(pos===6){
            jungleExample.rvAddMarker("Gank Mid or Top","yellow",627.5,465,false);
            jungleExample.rvAddMarker("Top","Aqua",492.5,197.5,false);
            jungleExample.rvAddMarker("Mid","Aqua",955,662.5,false);
            jungleExample.rvMoveChampion(jgler,490,262.5,false)
        }
    }

Quiz Functionality Example

Quiz Time!

You are playing vayne and your team has just won a team fight in the dragon pit. You aced the enemy team but sadly your top laner has died in the fight. There are 4 of you alive. You have 3 items and your midlaner also has 3 items. Volibear still has smite up. What do you do next? Drag your champion (vayne) to the location that you should be after this team fight.

Quiz Template

This library has a built in quiz functionality. This is great for educational web apps about League of Legends. The developer sets up a "correct zone" for champions and the user must drag one or several champions to their respective zones. This is great for quizzing users on their knowledge of the game because you can test them on what to do at a certain point in the game.

Web apps like gamerzclass.com specialize in teaching people how to play the game and offer lessons. This library functionality for quizzes can be used here to test users on their macro knowledge.

JavaScript Code - Creating A Quiz

const gameStateObject = '{"champions":[{"name":"Vayne","team":"blue","x":1080,"y":870,"movable":true},{"name":"Braum","team":"blue","x":1235,"y":770,"movable":false},{"name":"Volibear","team":"blue","x":1037.5,"y":730,"movable":false},{"name":"Anivia","team":"blue","x":892.5,"y":732.5,"movable":false}],"monsters":[{"name":"baron"}],"structures":[{"name":"bmt2"},{"name":"bmt3"},{"name":"btt1"},{"name":"btt2"},{"name":"btt3"},{"name":"bbt3"},{"name":"bn1"},{"name":"bn2"},{"name":"rmt2"},{"name":"rmt3"},{"name":"rtt2"},{"name":"rtt3"},{"name":"rbt2"},{"name":"rbt3"},{"name":"rn1"},{"name":"rn2"},{"name":"rti"},{"name":"rmi"},{"name":"rbi"},{"name":"bti"},{"name":"bmi"},{"name":"bbi"}],"markers":[{"text":"Won Team Fight At Dragon","x":1195,"y":977.5,"color":"violet","movable":false}]}';

//remove user deleting permission

//quiz set up

quizMap.rvSetQuizZone(quizMap.rvGetSpawnedChampions()[0],595,339,891,576);

//using jQuery
function evaluateQuiz(){
    if(quizMap.rvEvaluateQuiz()===true){
        $("#answerBox").text("Correct! You should do baron here.")
    }
    else{
        $("#answerBox").text("Incorrect! Press show answer if you are stuck!")
    }
}
